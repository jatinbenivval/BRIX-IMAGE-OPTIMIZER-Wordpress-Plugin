== Description ==

BRIX Image Optimizer is a high-performance WordPress plugin designed to seamlessly convert JPG and PNG images to the modern WebP format. Built with a focus on speed, server safety, and clean file management.

**💎 100% FREE FOREVER - No Hidden Fees!**

= Key Features =
* **Hybrid Optimization**: Automatic runtime generation + powerful bulk processing tool.
* **Smart Mirroring**: Keeps your main uploads folder clean by storing optimized assets in `/brix-optimized/`.
* **Non-Destructive**: Never touches or modifies your original JPG/PNG files.
* **Live Stats**: Built-in dashboard to track disk space saved.
* **Safe Lifecycle**: Reverts instantly on deactivation and wipes all data on uninstallation.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/brix-image-optimizer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->BRIX Optimizer screen to configure the plugin and start your first bulk scan.

== Frequently Asked Questions ==

= Does it modify my original images? =
No. BRIX is non-destructive. It creates a mirror folder and leaves your original JPG/PNG files untouched.

= What happens if I deactivate the plugin? =
Your site will instantly fallback to serving the original JPG/PNG images.
